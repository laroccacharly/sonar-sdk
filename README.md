# Sonar SDK

A Python SDK/CLI for interacting with Perplexity AI's Sonar models.

## Install
   - Create a `.env` file in the project root
   - Add your API key: `PLEX_API_KEY=your_api_key_here`

```
uv tool install -e .
```

## Usage

   ```
   plex -h # for help 
   plex "what is the weather in NYC?" 
   ```
